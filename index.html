About

IAIONEX is a sovereign, deterministic, fully offline AI system engineered from scratch by Johan Gärtner (IAIONEX.AB). It runs on Termux/ARM64, Linux and Windows with no cloud, no telemetry, and no external models.

What “built from scratch” really means

  • Original architecture: every core component is designed and implemented independently — not a wrapper around external frameworks.
  • Sovereign control: runs locally under the owner’s authority; nothing phones home.
  • Deterministic results: the same input always yields the same output, enabling audit and reproducibility.
  • UID‑locked ownership: artifacts are bound to JOHAN GÄRTNER • IAIONEX.AB • UID=IAIONEX.70ID for traceable authorship.
  • Cryptographic integrity: files can be verified with SHA‑256 and optional QR tags.

Technology

  • Token‑free text generation: coherent long‑form output by deterministic composition instead of neural token sampling.
  • Planner‑IR (explainable reasoning): goals, steps and evidence represented in a readable plan so decisions are inspectable.
  • Mathematical core: precise arithmetic and proof tokens with explicit bounds for dependable results.
  • WFST output shaping: deterministic formatting for headings, lists and tables — no repetition or drift.
  • Append‑only ledger: tamper‑evident history with replay for audits and forensic review.
  • Sealed exports: ODT/PDF/TAR packages carry UID + SHA‑256 + optional QR for instant authenticity checks.
  • Sentinel security: watchdogs and policy gates enforce offline operation and block injections.
